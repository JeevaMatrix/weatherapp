About this Project

Welcome to my Weather App project! This web application was created as a part of my learning journey into web development, specifically focusing on API integration and creating dynamic, user-friendly interfaces.

Key Features:

  • Current Weather Data: The app retrieves and displays real-time weather information such as temperature, precipitation rate, humidity, pressure, and wind speed.
  • 5-Day Forecast: You can also view the upcoming weather forecast for the next five days.
  • Hourly Forecast: There's a scrollable section that shows the forecast for each hour of the day, helping you stay prepared for any weather changes.
  • Location-Based Data: The app automatically retrieves weather data based on the user's location and updates the display in real-time when the page is refreshed.
  • Smooth Scrolling Navigation: The navigation links offer smooth transitions between different sections of the page for a better user experience.

Technologies Used:

  • HTML/CSS/JavaScript: For building the structure, style, and interactivity of the website.
  • OpenWeather API: This project uses the OpenWeather API to fetch all the necessary weather data and forecast information.
  • Flexbox & Grid Layouts: To create responsive designs that adjust seamlessly on different screen sizes.
  • Responsive Design: The app adapts well on both mobile and desktop screens, offering a consistent experience across devices.

Experience:

I thoroughly enjoyed working on this project and learned a lot throughout the development process. I had the opportunity to dive deeper into API handling, weather data integration, and improving UI/UX design. MSN Weather served as a primary inspiration for the layout, helping me craft a visually appealing and easy-to-use interface.

This project helped me enhance my web development skills, particularly in API integration, CSS animations, and JavaScript functionality. Overall, it was a rewarding experience that gave me practical insights into building dynamic web applications.
